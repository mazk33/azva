# AZVA Motion System

## Motion Principle

AZVA does not animate. It settles.

Motion should feel like:
- Systems aligning
- Information emerging
- Decisions becoming clear
- Momentum forming quietly

## Preferred Motion

- Fade in
- Fade up
- Gentle hover lift
- Subtle arrow movement
- Smooth opacity transitions

## Avoid

- Bounces
- Spins
- Zooms
- Flashy parallax
- Aggressive page transitions
- Overlapping motion
- Decorative animation

## Timing

- Micro-interactions: 180–260ms
- Section entrances: 500–700ms
- Hover states: 200–300ms

## Easing

Use calm ease-out transitions.

Motion should be almost invisible.
