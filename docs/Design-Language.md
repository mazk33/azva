# AZVA Design Language

## Design Principle

Editorial clarity meets operational intelligence.

## Visual Philosophy

AZVA should feel composed, intelligent, and enduring.

The design should communicate:
- Alignment
- Structure
- Systems
- Clarity
- Movement with discipline

## Core Visual System: Alignment Geometry

AZVA visuals are built from five primitives:

1. Point
2. Line
3. Grid
4. Node
5. Plane

These primitives can create:
- Strategic diagrams
- Operational maps
- PRIZM visualizations
- Venture ecosystems
- Presentation graphics
- Report visuals

## Visual Rules

Prefer:
- Large whitespace
- Strong typography
- Restrained color
- Thin lines
- Subtle grids
- Calm geometric structure

Avoid:
- Stock photography
- Generic AI graphics
- Glowing brains
- Circuit boards
- Handshakes
- Office photography
- Excessive gradients
- Decorative icons

## Color System

Primary:
- Midnight: #1E2A38

Accent:
- Warm Clay: #B56E46

Canvas:
- Ivory: #F8F4EE

Secondary:
- Moss: #6D8B74

Text:
- Carbon: #0E0E0E

## Color Usage

- Ivory should dominate.
- Midnight creates authority.
- Warm Clay creates energy.
- Moss supports calm growth.
- Carbon is for precision.

Color should guide attention, not decorate.
