# AZVA Design System Polish Pack

## Purpose

This pack standardizes motion, cards, buttons, institutional links, focus states,
and reduced-motion behavior without changing page architecture.

## Motion vocabulary

- `FadeIn`: standard editorial reveal
- `Reveal`: slower chapter-level reveal
- `Lift`: restrained interactive elevation
- `Stagger`: parent for grouped content
- `StaggerItem`: child within a staggered group
- `MotionCard`: shared interactive surface
- `MotionLink`: subtle directional movement for text links

## Recommended first integrations

1. Homepage "How We Think" principle cards
2. PRIZM metric cards
3. Ventures selection cards
4. Perspectives recent entries
5. Connect topic tiles

Use stagger only for grouped items entering the viewport together.
